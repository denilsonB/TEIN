Bootstrap 4 Input

Bootstrap suporta todos os tipos HTML5 entrada: texto, senha, data e hora, data e hora-local, data, mês, hora, semana, número, email, url, busca, tel, e cor.

O exemplo que se segue contém dois elementos de entrada; um do tipo "text" e um do tipo="password"
<div class="form-group">
   <label>Nome: </label>
   <input type="text" class="form-control">
</div>
<div class="form-group">
   <label>Senha: </label>
   <input type="password" class="form-control">
</div>
