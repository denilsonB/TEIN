O bootstrap 4 providencia os seguintes estilos de botões :

  • <button type="button" class="btn">Basic</button>

    com a classe 'btn' o botão possui um fundo transparente
  • <button type="button" class="btn-primary"> Primary </button>

    com o 'btn-primary' o fundo é azul
  • <button type="button" class="btn-secondary"> Secondary </button>

    com o 'btn-secondary' o fundo é cinza
  • <button type="button" class="btn-success"> Success </button>

    com o 'btn-success' o fundo é verde
  • <button type="button" class="btn-info"> Info </button>

    com o 'btn-info o fundo' é azul-claro
  • <button type="button" class="btn-warning">Warning </button>

    com o 'btn-warning' o fundo é Amarelo
  • <button type="button" class="btn-danger">danger </button>

    com o 'btn-danger' o fundo é Vermelho
  • <button type="button" class="btn-dark">dark </button>

    com o 'btn-dark' o fundo é Preto
  • <button type="button" class="btn-light">light </button>

    com o 'btn-light' o fundo é Branco
  • <button type="button" class="btn-link">link </button>

    O 'btn-link' é utilizado para colocar um link
