O bootstrap utiliza o estilo 'classe-valor', a classe para os botões é a btn:

  • <button type="button" class="btn">Basic</button>

    se não colocar o valor por padrão o btn tem fundo transparente
  • <button type="button" class="btn-primary"> Primary </button>

    Com o valor 'primary' o fundo fica azul
  • <button type="button" class="btn-secondary"> Secondary </button>

    Com o valor 'secondary' o fundo fica cinza
  • <button type="button" class="btn-success"> Success </button>

    Com o valor 'success' o fundo fica verde
  • <button type="button" class="btn-info"> Info </button>

    Com o valor 'info' o fundo fica Azuk claro
  • <button type="button" class="btn-warning">Warning </button>

    Com o valor 'warning' o fundo fica amarelo
  • <button type="button" class="btn-danger">danger </button>

    Com o valor 'danger' o fundo fica vermelho
  • <button type="button" class="btn-dark">dark </button>

    Com o valor 'dark' o fundo fica preto
  • <button type="button" class="btn-light">light </button>

    Com o valor 'light' o fundo fica branco
  • <button type="button" class="btn-link">link </button>

    Com o valor 'link' passa um link no botão
